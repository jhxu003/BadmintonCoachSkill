# Annotation Guide

Annotate only observable facts. Do not infer invisible biomechanics from weak video evidence.

## Required Observation Classes

- Contact point: `front_high`, `behind_head`, `low`, `unknown`.
- Elbow height before hit: `below_shoulder`, `near_shoulder`, `above_shoulder`, `missing`.
- Wrist/elbow sequence: `elbow_before_wrist`, `wrist_before_elbow`, `missing`.
- Hip/shoulder sequence: `hip_before_arm`, `late_hip`, `missing`.
- Racket-side structure: `stable`, `collapsed`, `missing`.
- Follow-through: `complete`, `short`, `missing`.
- Footwork arrival timing: `on_time`, `late`, `missing`.
- Recovery: `normal`, `slow`, `missing`.

When a feature is not visible, mark it as missing rather than guessing.
