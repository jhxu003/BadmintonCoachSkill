# Project Positioning

BadmintonCoachSkill is a knowledge-layer project for badminton coaching agents.

It focuses on turning structured observations into coaching decisions:

```text
student profile + video observation -> framework -> issue priority -> drill -> retest
```

It is not a raw-video CV model and not an official Liu Hui product.

## Target Users

- Developers building badminton video-analysis agents.
- Coaches who want a structured report draft format.
- Researchers studying skill/rubric-driven sports feedback.

## V1 Boundary

- In scope: high clear, smash, rear-court footwork, student-fit framework selection.
- Out of scope: pose estimation, shuttle tracking, full-match tactics, medical diagnosis.
